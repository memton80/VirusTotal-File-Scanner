Clé API VirusTotal
Ta clé est stockée localement dans ton navigateur et chiffrée par Firefox
Comment obtenir une clé API ?
  1. Crée un compte gratuit sur VirusTotal.com
  2. Va dans ton profil → API Key → API Key
  3. Copie ta clé API (64 caractères)
  4. Colle-la dans le champ ci-dessus
Limite de l'API gratuite
Compte gratuit: 4 requêtes par minute, 500 requêtes par jour
Taille maximale: 32 MB par fichier

Cette extension respecte automatiquement ces limites. Pour un usage intensif, consulte les plans payants.
Sécurité & Confidentialité

Protection des données :

  • Ta clé API est stockée localement et chiffrée par Firefox
  • Toutes les communications utilisent HTTPS (TLS 1.3)
  • Aucune donnée n'est envoyée à des tiers (sauf VirusTotal)
  • Le code source est ouvert sur GitHub
  • Validation stricte de toutes les entrées utilisateur

Important: Les fichiers uploadés deviennent publics sur VirusTotal. Ne scanne pas de fichiers confidentiels !
